# 프로젝트기반 프론트엔드 개발자 양성과정 : 프로젝트 저장소
훈련 과정 중에 수행한 프로젝트의 코드를 모아둔 레파지토리입니다.

# 01.웹 프로젝트
HTML,CSS,JavaScript만을 사용하여 웹사이트를 구현한 프로젝트입니다.

# 02.모바일 웹&앱 프로젝트
뷰를 활용하여 웹 어플리케이션을 만드는 프로젝트이며 일부는 SPA로 구현되도록 하였습니다.

# 03.SPA 프로젝트 (citadellegin-app)
